# Laravel WSL Bootstrap Script

This script installs most Laravel development dependencies automatically.

---

# Create Script

nano laravel-wsl-bootstrap.sh

Paste script and save.

---

# Make Executable

chmod +x laravel-wsl-bootstrap.sh

---

# Run Script

./laravel-wsl-bootstrap.sh

---

# Verify Installation

php -v

composer -V

node -v

mysql --version

nginx -v
