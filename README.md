# WSL Laravel Development Setup Guide

This repository provides a complete guide for setting up a Laravel development environment using WSL.

The environment supports running multiple Laravel versions from Laravel 8 up to Laravel 12 without dependency conflicts.

Technologies covered:

- WSL2
- Ubuntu
- PHP (multiple versions)
- Composer
- Node.js
- MySQL
- nginx
- asdf version manager

This setup is designed for developers working on multiple Laravel projects that require different PHP versions.
