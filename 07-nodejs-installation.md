# Node.js Installation

Laravel uses Node.js for frontend tooling such as:

- Vite
- npm
- asset compilation
- frontend frameworks

---

# Install Node.js

Install Node.js 20:

curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -

sudo apt install -y nodejs

---

# Verify Installation

node -v

npm -v

Example output:

v20.x.x

---

# Why Node Is Needed

Laravel uses Node for:

- compiling CSS
- compiling JavaScript
- running Vite
- building frontend assets
