# Laravel Version Compatibility

Different Laravel versions require different PHP versions.

---

# Laravel vs PHP

Laravel 12 → PHP 8.2+  
Laravel 11 → PHP 8.2+  
Laravel 10 → PHP 8.1+  
Laravel 9 → PHP 8.0+  
Laravel 8 → PHP 7.3 – 8.1

---

# Example

Project A:

Laravel 12

Use:

PHP 8.2 or 8.3

Project B:

Laravel 8

Use:

PHP 8.1
