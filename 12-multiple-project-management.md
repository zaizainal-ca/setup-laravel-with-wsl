# Managing Multiple Laravel Projects

When working with multiple projects it is important to keep them organized.

---

# Recommended Directory

~/projects

---

# Example Structure

~/projects

laravel12-app  
laravel10-api  
laravel8-legacy

---

# Avoid Windows Filesystem

Do NOT store projects inside:

/mnt/c/

because performance is slower.

Always use:

/home/username/projects
